Registers wellcut.tv oEmbed provider with WordPress and adds it to the whitelist

== Description ==
Registers wellcut.tv oEmbed provider with WordPress and adds it to the whitelist, allowing all users who can write posts to use this provider.

== Installation ==
Use WordPress' Add New Plugin feature, searching "oEmbed Wellcut", or download the archive and:

Unzip the archive on your computer
Upload duplicate-post directory to the /wp-content/plugins/ directory
Activate the plugin through the 'Plugins' menu in WordPress
